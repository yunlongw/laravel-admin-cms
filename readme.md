## install

``` shell

composer-install

npm install 

// 生成 laravel key
php artisan key:gen

// 生成jwt key
php artisan jwt:secret

//安装 laravel-echo-server 
npm install -g laravel-echo-server

//启动 laravel-echo-server , 修改配置  laravel-echo-server.json
laravel-echo-server start
```
